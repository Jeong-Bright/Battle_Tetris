# 전투 테트리스 (Battle Tetris)
테트리스 코드 개선 프로젝트

## 추가 및 개선할 기능
1. 배경음, 효과음

2. 디자인 요소 추가(블록 이미지, 배경화면 등등)

3. 코드 내 랜덤 클래스 변경

4. 메인 화면
   * 게임 시작(대결 모드 선택)
   * 게임 설명(조작법)

5. 대결 모드
   * 플레어어 vs 플레이어 (단일 컴퓨터 2p)
   * 플레이어 vs AI

6. 아이템
   * 폭탄 블록
   * hp 회복
   * 장애물 블록 소환
